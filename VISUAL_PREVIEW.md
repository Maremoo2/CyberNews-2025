## Visual Design

### Color Themes

**NIST (Teal/Green):**
- Main border: `#16a085` (teal)
- Introduction: Teal gradient background
- Core functions: Light green backgrounds
- Subsections: Various complementary colors

**DORA (Orange):**
- Main border: `#e67e22` (orange)
- Introduction: Orange gradient background
- Main pillars: Light orange backgrounds
- Subsections: Various complementary colors

### Layout

- **Responsive grid layouts** for all subsections
- **Card-based design** for easy scanning
- **Color-coded sections** for visual organization
- **Icons and emojis** for quick identification
- **Mobile-friendly** with single-column layout on small screens

### Typography

- **Clear hierarchy** with H3, H4, H5 headings
- **Easy-to-read fonts** with good line height
- **Highlighted information** in colored boxes
- **Lists and bullets** for structured information

---

## User Experience

Users will experience:

1. **Easy Discovery**: Regulation cards appear in the main grid
2. **Quick Overview**: Cards provide essential information at a glance
3. **Deep Dive**: Detailed sections offer comprehensive information
4. **Resource Access**: Direct links to official documents and guides
5. **PDF Placeholders**: Clear indication where to add PDF documents
6. **Consistent Design**: Same visual language as existing regulations
7. **Mobile Responsive**: Works perfectly on all device sizes

This implementation matches the quality and comprehensiveness of the NIS2 section!
