# NIST Cybersecurity Framework Documents

This folder contains official documents related to the NIST Cybersecurity Framework.

## Documents to be added:

1. **NIST-CSF-2.0.pdf** - NIST Cybersecurity Framework version 2.0
2. **NIST-implementation-guide.pdf** - Implementation guide for NIST CSF

## External Resources:

- Official NIST CSF 2.0: https://www.nist.gov/cyberframework
- NIST CSF Documentation: https://csrc.nist.gov/projects/cybersecurity-framework
- NIST Special Publications: https://csrc.nist.gov/publications

## How to add PDFs:

Place your PDF files in this directory and they will be accessible via:
- `/documents/nist/your-filename.pdf`

## Note:
PDFs are served from the public directory and will be available at runtime.
