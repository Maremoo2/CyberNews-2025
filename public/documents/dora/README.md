# DORA (Digital Operational Resilience Act) Documents

This folder contains official documents related to DORA - EU Regulation 2022/2554.

## Documents to be added:

1. **DORA-regulation-norwegian.pdf** - Norwegian language version of DORA regulation
2. **DORA-regulation-english.pdf** - English language version of DORA regulation

## External Resources:

- Official EU Legal Text: https://eur-lex.europa.eu/legal-content/EN/TXT/?uri=CELEX:32022R2554
- Finans Norge DORA Information: https://www.finansnorge.no/tema/digitalisering/eus-digitale-agenda/forordningen-om-digital-operasjonell-motstandskraft---dora/
- PwC DORA Guide (Norwegian): https://www.pwc.no/no/innsikt/dora-alt-du-trenger-a-vite.html

## How to add PDFs:

Place your PDF files in this directory and they will be accessible via:
- `/documents/dora/your-filename.pdf`

## Note:
PDFs are served from the public directory and will be available at runtime.
