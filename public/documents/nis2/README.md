# NIS2 Directive Documents

This folder contains official documents related to the NIS2 Directive (EU) 2022/2555.

## Documents to be added:

1. **NIS2-direktiv-norsk.pdf** - Norwegian language version of NIS2 directive
2. **NIS2-directive-english.pdf** - English language version of NIS2 directive

## External Resources:

- Official EU Legal Text: https://eur-lex.europa.eu/legal-content/EN/TXT/HTML/?uri=CELEX:52020PC0823
- Norwegian Government EEA Note: https://www.regjeringen.no/no/sub/eos-notatbasen/notatene/2021/feb/nis2-direktivet/id2846097/

## How to add PDFs:

Place your PDF files in this directory and they will be accessible via:
- `/documents/nis2/your-filename.pdf`

## Note:
PDFs are served from the public directory and will be available at runtime.
